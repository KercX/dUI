WinCleaner Pro Ultimate
⚡ Advanced Windows cleaning, optimization, and maintenance toolkit built with Batch, PowerShell, and optional Python GUI.

✨ Overview
WinCleaner Pro Ultimate is a lightweight yet powerful system utility designed to:
* Clean junk and temporary files 🧹
* Improve system performance ⚡
* Provide quick access to driver updates 🔄
* Display detailed system information 💻
* Perform deep cleaning using PowerShell 🧠
This project is fully open-source and designed for learning, customization, and real-world usage.

🔥 Features
🧹 Cleaning Engine
* Remove TEMP files
* Clean Prefetch data
* Clear Recycle Bin
* Flush DNS cache
* Remove logs and thumbnails (deep clean)
🔄 Driver Update Access
* Quick launch to Windows Update
* Helps keep drivers up to date manually
💻 System Information
* Full system diagnostics
* Disk usage and free space overview
⚡ Deep Clean Mode
* Advanced cleanup via PowerShell
* Removes hidden cache and logs
🧠 Interactive Menu
* Simple CLI interface
* Fast navigation
* Beginner-friendly
🐍 Optional GUI
* Python-based interface
* Expandable for future versions

🪟 Windows Support
✅ Supported Versions
* Windows 10 (all editions)
* Windows 11 (all editions)
⚙️ Requirements
* Administrator privileges
* PowerShell enabled
* Windows Update service active
❗ Important Notes
* Some features require Run as Administrator
* Deep cleaning uses system-level access
* Works best on clean, non-restricted systems
🚫 Not Supported
* Windows 7 / 8 (may partially work, not tested)
* Linux / macOS (use shell alternatives instead)

⚙️ Installation
📥 Method 1 — Git Clone
git clone https://github.com/your-username/WinCleaner-Pro-Ultimate.git
cd WinCleaner-Pro-Ultimate
📦 Method 2 — Download ZIP
1. Click Code → Download ZIP
2. Extract files
3. Open folder

▶️ Usage
Run the main menu:
menu.bat
🔐 IMPORTANT
➡️ Right-click → Run as Administrator

🧪 CI/CD Integration
This project uses GitHub Actions for:
* Script validation
* Automated checks
* Basic testing
Files:
* .github/workflows/main.yml
* .github/workflows/test.yml

🐍 Python GUI (Optional)
Install dependencies:
pip install -r requirements.txt
Run GUI:
python src/python/cleaner_gui.py

⚠️ Safety & Disclaimer
* This tool modifies system-level files
* Always run with caution
* Do NOT delete unknown system directories
* Backup recommended before deep clean
